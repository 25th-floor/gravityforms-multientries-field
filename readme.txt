Enhance your Gravitiyforms with bulk entry support

== Description ==

With this plugin for Gravityforms you can choose a form section to be a bulk section.
This section can be cloned by the user as often as he wants and all data is
validated and saved. There is also a possibility to style the data to send it via mail.

Note: Plugin development was sponsored by bor!sgloger @ http://borisgloger.com

== Installation ==

1. Install via plugin directory or upload the package contents to your `/wp-content/plugins/` directory.

== Usage ==

Note: Don't forget to activate this plugin!

1. Add a hidden field with the name 'bulk' inside the section you want to support.
1. That's it!

Note: Currently only one bulk field per form is supported!
